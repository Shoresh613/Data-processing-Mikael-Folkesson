Frontend vs backend

Frontend: The client-side part of an application that focuses on presenting stuff to the end-user

Backend: The server-side part of an application that focuses on on the behind-the-scenes, logic, computations , and data storage.

3 basic web technologies:

  • HTML: Markup language that describes the laoyut and content of a static web page.
  • JavaScript: Client side scripting language used to add dynamic functionality to a webpage.
  • CSS: A language used to style a webpage.
The browser interprets the HTML, CSS, and JavaScript code to display the webpage, and constructs a DOM tree structure, through which the elements of the page can be accessed.
